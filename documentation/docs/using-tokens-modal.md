# Using Tokens Modal

## Usage on a given Page

Finally, use the `SelectToken` component in one of your pages. Import the component and render it.

```tsx
import SelectToken from './SelectToken'

const SwapPage = () => {

    return (
        <div>
            <SelectToken />
        </div>
    )
}
```
